Setup Required

This application requires a Firebase configuration to function. Please follow these steps to set it up:

  1. Create a new Firebase project.
  2. Add a new web app to your project.
  3. Copy the `firebaseConfig` object provided by Firebase.
  4. Paste it into the `script` tag below the comment "PASTE YOUR FIREBASE CONFIG HERE".

Once you've done this, the chat will connect and start working.
